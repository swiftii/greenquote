Lawn Care Quote Widget

Production-ready widget for GreenQuote Pro

How to Embed This Widget

Copy and paste this iframe code anywhere on your website:

<iframe src="https://app.getgreenquote.com/widgets/lawn/v1/index.html?client=example-lawn" style="width:100%; max-width:480px; height:850px; border:0;" loading="lazy" ></iframe>

Replace:

  • client=example-lawn - Your client config file name

Works perfectly with GoHighLevel, WordPress, Wix, Squarespace, and any platform that supports HTML!
